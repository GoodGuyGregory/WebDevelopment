## Basics of Making sites interactive  

  * Selecting elements on the page

## Events  

Basic events check [MDN](https://developer.mozilla.org/en-US/docs/Web/Events)

**Laptop/Desktops**  

  * click
  * dblclick  
  * mousedown  
  * mouseup  
  * mousemove  
  * mouseover  
  * mouseout  

**SmartPhones/Tablets**  

  * touchstart
  * touchmove  
  * touchend
