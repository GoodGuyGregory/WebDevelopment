## Vim 

**Basic Operations**  

Getting into Vim  

`vim` of `vim <file>`

quiting Vim `:q` or `:q!`

**Opening Files**  

inside vim type `:e` and you can add the directory of a file...

**Writing a File**

`:w` saves a file or writes to a file.

**Movement** 

`h j k l` Up Down Left Right or Arrow Keys
